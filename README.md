# European Airline Route Network Analysis

This project presents a social network analysis of the European airline transportation network using graph theory and NetworkX.

The network is constructed from real-world European airline route data, where airports are represented as nodes and direct flight routes as edges. Various network analysis techniques are applied to identify the structural properties and the most influential airports within the network.

## Features

- Data Cleaning & Exploratory Data Analysis (EDA)
- Directed and Undirected Network Construction
- Degree Centrality Analysis
- In-Degree & Out-Degree Analysis
- Betweenness Centrality
- Closeness Centrality
- PageRank
- Clustering Coefficient
- Community Detection
- Network Visualization

## Technologies

- Python
- Pandas
- NetworkX
- Matplotlib
- Jupyter Notebook

## Dataset

The project uses a real-world European airline routes dataset containing airports, flight routes, flight frequencies, ticket prices, and airport information.

## Objective

The objective is to investigate the topology of the European air transportation network and identify key hub airports using social network analysis techniques.
